!!!!!!!!!!!!!!!!!!!!!!!!
!     REQUIREMENTS     !
!!!!!!!!!!!!!!!!!!!!!!!!
1. JDK 1.5 (or higher)
2. JRE 1.5 (or higher)
3. JUnit 4
4. JDepend (optional)
5. JavaDoc (optional)
6. A Java IDE like Eclipse, NetBeans, or IntelliJ. An IDE with plug-in for JUnit and JDpend is recommended


***************************
 HOW TO RUN PROGRAM IN IDE
***************************
1. Make sure you have the required requirements mentioned in previous section.

2. Open IDE. Import this project into your IDE.
   For Eclipes user, this is done by New -> Java Project -> Write "Virtumon" in Project Name -> Finish.

3. Open file Main.java located at Virtumon/src/main/Main.java.

4. Run the program.

5. Now you can play the game.


********************************
 HOW TO RUN PROGRAM IN TERMINAL
********************************
1. Make sure you have the 1st and 2nd requirements.

2. Open folder Virtumon, you should see a file named Virtumon.jar.

3. Now, open terminal and change the directory to Virtumon directory.

4. Execute command
	java -jar Virtumon.jar

5. Now you can play the game.

****************************
 HOW TO RUN JUNIT TEST CASE
****************************
1. Open the IDE and project Virtual Zoo. Then open the content of AllTest.java located in Virtumon/main/AllTest.java

2. Run the program as JUnit Test

 HOW TO GENERATE JAVADOC
*************************
1. Open project VirtualZoo. Select menu Windows and find Generate Javadoc (in Eclipse)


*********************
 HOW TO SHOW JDEPEND
*********************
1. Open JDpend windows in your IDE. Select the package which you want to check.
